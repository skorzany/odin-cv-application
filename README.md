# odin-cv-application

A simple CV/resumee generator created in React (with Vite). It was a good opportunity
to put my newly-acquired skills into practice (especially <em>props</em> and <em>state</em>).

The generated CV consists of three sections:

- Contact section (required), allows user to add and edit their contact info,
- Education section, allows users to add, edit and delete their education history,
- Work experience section, allows same functionality as Education section, but uses
  different form elements.

After submitting the form, all input is gathered and alerted in the browser,
and default (empty) state is restored. The data is not sent or stored anywhere.

I hope that you like it!
